# Wireguard Manager

A simple program to manage multiple wireguard interfaces and peers.

# Build

```bash
sudo apt install -y gcc make libjson-c-dev git;
git clone https://github.com/ammarfaizi2/wg-mgr.git;
cd wg-mgr;
make -j4;
```
